# Experiment Design Workflow

## Overview

Step-by-step process for designing rigorous experiments from research question to protocol finalization.

## Workflow Steps

### Phase 1: Question Development (1-2 weeks)

**1. Identify Research Gap**
- Review existing literature
- Identify contradictions or unknowns
- Assess significance and novelty

**2. Formulate Research Question**

Use: [Research Design Guide](../guides/research-design.md)

Apply FINER criteria:
- [ ] **Feasible**: Resources, time, expertise available?
- [ ] **Interesting**: Matters to field?
- [ ] **Novel**: Fills genuine gap?
- [ ] **Ethical**: Can be conducted ethically?
- [ ] **Relevant**: Results will be useful?

Make it specific (PICO):
- **P**opulation: Who will be studied?
- **I**ntervention: What treatment/exposure?
- **C**omparison: Compared to what?
- **O**utcome: What will be measured?

**Example:**
- Vague: "Does exercise help depression?"
- Specific: "In adults with major depressive disorder (P), does 12-week aerobic exercise program (I) compared to usual care (C) reduce depressive symptoms measured by BDI-II (O)?"

**3. Formulate Hypotheses**
- State as testable predictions
- Primary hypothesis (main question)
- Secondary hypotheses (additional questions)
- Specify directionality if appropriate

---

### Phase 2: Design Selection (1 week)

**4. Choose Study Design**

Use: [Research Design Guide](../guides/research-design.md)

Match design to question:
- **Causal question** → RCT or quasi-experimental
- **Associational** → Cohort, case-control, cross-sectional
- **Descriptive** → Survey, case series

Design choices:
- [ ] Between-subjects vs. within-subjects
- [ ] Parallel groups vs. crossover
- [ ] Factorial design (if multiple factors)
- [ ] Number of treatment arms
- [ ] Duration of intervention/follow-up

**5. Plan Bias Minimization**

Use: [Bias Detection Guide](../guides/bias-detection.md)

**Randomization plan:**
- [ ] Random sequence generation method
- [ ] Allocation concealment procedure
- [ ] Stratification variables (if any)
- [ ] Block randomization (if applicable)

**Blinding plan:**
- [ ] Can participants be blinded? How?
- [ ] Can providers be blinded? How?
- [ ] Can outcome assessors be blinded? How?
- [ ] Blinding integrity checks planned?

**Control group:**
- [ ] Placebo control
- [ ] Active control (standard treatment)
- [ ] Wait-list control
- [ ] No-treatment control

**Confound control:**
- [ ] Key confounders identified
- [ ] Measured at baseline
- [ ] Control by design (matching, restriction)
- [ ] Control by analysis (adjustment)

---

### Phase 3: Sample Planning (1 week)

**6. Power Analysis**

Calculate required sample size:
1. Specify expected effect size (from literature or pilot)
2. Set desired power (typically .80)
3. Set alpha level (typically .05)
4. Calculate n per group

Account for attrition:
- Estimate dropout rate
- Inflate sample accordingly
- Plan interim monitoring

**7. Sampling Strategy**

Define population:
- [ ] Target population (who you want to generalize to)
- [ ] Accessible population (who you can recruit from)
- [ ] Inclusion criteria (who is eligible)
- [ ] Exclusion criteria (who is not eligible)

Choose sampling method:
- Random sampling (ideal but often infeasible)
- Stratified sampling
- Cluster sampling
- Convenience sampling (weakest)

**8. Recruitment Plan**
- Where will participants be recruited?
- What recruitment materials?
- What incentives (if any)?
- Estimated recruitment rate
- Estimated timeline

---

### Phase 4: Measurement Planning (1-2 weeks)

**9. Select Measures**

Use: [Methodology Critique Guide](../guides/methodology-critique.md)

For each outcome:
- [ ] Primary outcome clearly designated
- [ ] Validated instruments selected
- [ ] Reliability data available (α, ICC, κ)
- [ ] Validity evidence reviewed
- [ ] Sensitive to expected change?
- [ ] Multiple measures for key constructs (triangulation)

**10. Operationalize Variables**

For each variable:
- Conceptual definition (what it means)
- Operational definition (how you'll measure it)
- Measurement procedure (step-by-step)
- Scoring procedure
- Normal range / cutoffs

**11. Data Collection Procedures**
- When will each measure be administered?
- Who will collect data?
- Training procedures for data collectors
- Quality control procedures
- Data storage and security

---

### Phase 5: Analysis Planning (1 week)

**12. Statistical Analysis Plan**

Use: [Statistical Evaluation Guide](../guides/statistical-evaluation.md)

Pre-specify:
- [ ] Primary analysis (matching primary outcome)
- [ ] Secondary analyses
- [ ] Subgroup analyses (with interaction tests)
- [ ] Sensitivity analyses
- [ ] Missing data handling approach
- [ ] Multiple comparison correction
- [ ] Assumption checks

**13. Sample Description Plan**
- Baseline characteristics (Table 1)
- Participant flow (CONSORT diagram)
- Attrition analysis

**14. Effect Size Calculation**
- Which effect sizes will be reported?
- Confidence interval calculations

---

### Phase 6: Ethics and Registration (2-4 weeks)

**15. Ethics Approval**
- Prepare IRB/ethics application
- Informed consent procedures
- Risk mitigation plans
- Data protection plans
- Adverse event reporting

**16. Pre-registration**
- Register at ClinicalTrials.gov or OSF
- Specify all hypotheses
- Specify all planned analyses
- Upload analysis plan
- Time-stamp before data collection

---

### Phase 7: Protocol Finalization (1 week)

**17. Write Protocol**

Include:
- [ ] Background and rationale
- [ ] Research question and hypotheses
- [ ] Study design
- [ ] Participants (eligibility, recruitment, sample size)
- [ ] Randomization and blinding procedures
- [ ] Interventions (detailed description)
- [ ] Outcomes (measures and procedures)
- [ ] Statistical analysis plan
- [ ] Ethics and informed consent
- [ ] Timeline
- [ ] Data management plan

**18. Pilot Test**
- Test procedures on small sample
- Identify procedural issues
- Refine protocols
- Verify feasibility

**19. Finalize Materials**
- Informed consent forms
- Data collection forms
- Training materials
- Randomization materials
- Blinding procedures

---

## Design Checklist

### Research Question
- [ ] Clear and specific
- [ ] Testable and falsifiable
- [ ] FINER criteria met
- [ ] PICO specified (if applicable)

### Study Design
- [ ] Design matches question type
- [ ] Design justified
- [ ] Comparison groups appropriate
- [ ] Duration/timing justified

### Bias Minimization
- [ ] Randomization planned
- [ ] Allocation concealment planned
- [ ] Blinding maximized
- [ ] Control group(s) appropriate
- [ ] Confounders identified and controlled

### Sample
- [ ] Power analysis completed
- [ ] Sample size adequate
- [ ] Attrition accounted for
- [ ] Inclusion/exclusion clear
- [ ] Recruitment plan feasible

### Measurement
- [ ] Primary outcome designated
- [ ] Validated instruments
- [ ] Multiple measures where appropriate
- [ ] Procedures standardized
- [ ] Assessor training planned

### Analysis
- [ ] Analysis plan pre-specified
- [ ] Primary analysis clear
- [ ] Multiple comparisons addressed
- [ ] Missing data plan specified
- [ ] Effect sizes and CIs planned

### Transparency
- [ ] Protocol written
- [ ] Pre-registered
- [ ] Ethics approved
- [ ] Reporting guidelines identified (CONSORT, etc.)

---

## Common Pitfalls to Avoid

1. ❌ Vague research question → ✓ Specific, testable question
2. ❌ No power analysis → ✓ A priori power analysis
3. ❌ Inadequate sample → ✓ Powered sample with attrition buffer
4. ❌ Poor randomization → ✓ Proper sequence generation + allocation concealment
5. ❌ No blinding when feasible → ✓ Maximum feasible blinding
6. ❌ Unvalidated measures → ✓ Validated, reliable instruments
7. ❌ No analysis plan → ✓ Pre-specified, registered plan
8. ❌ Confounding ignored → ✓ Confounders measured and controlled
9. ❌ No pre-registration → ✓ Public pre-registration
10. ❌ Ignoring ethics → ✓ IRB approval before starting
